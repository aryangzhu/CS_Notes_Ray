##### HTTML+CSS+JavaSrcipt即 页面结构+样式+脚本


１．基础


```html
<!--告诉浏览器我们使用什么协议-->
<!DOCTYPE html>
<html lang="en">
<!--head标签代表网页头部-->
<head>
<!--meta描述性标签,他用来描述我们网站的一些信息-->
<!--一般用来做SEO-->
<meta charset="UTF-8">
<meta name="keywords" content="狂神说Java,西部开源">
<meta name="description" content="这个地方可以学习Java">
<title>这是我第一个网页</title> <!--网页标题-->
</head>
<body>
    Hello,world!
</body>
</html>
```
